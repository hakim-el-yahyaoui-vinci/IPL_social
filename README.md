## Description du projet
Ce projet est une bibliothèque de validation d'emails réalisée en TypeScript.
Il utilise une approche TDD (Test Driven Development) et un pipeline CI avec GitHub Actions pour assurer la qualité du code.

## Comment lancer les tests
Pour exécuter la suite de tests, utilisez la commande suivante dans le terminal :

npm test
